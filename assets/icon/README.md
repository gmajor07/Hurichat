# App Icon

Place your app icon file here as `app_icon.png`

## Requirements:
- Format: PNG
- Size: At least 1024x1024 pixels (recommended)
- Background: Transparent or solid color
- Style: Simple, recognizable icon (ticket, cinema seat, etc.)
